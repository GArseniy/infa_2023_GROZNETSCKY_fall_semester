cat - concatenate files and print on the standart output
grep - searches  for  PATTERNS  in  each  FILE.   PATTERNS  is one or more
       patterns separated by newline characters, and grep prints each line  that
       matches a pattern.  Typically PATTERNS should be quoted when grep is used
       in a shell command.
head - Print the first 10 lines of each FILE to standard output.  With more than
       one FILE, precede each with a header giving the file name.
yes - Repeatedly output a line with all specified STRING(s), or 'y'.
